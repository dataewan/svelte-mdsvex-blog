Starting off

2021-03-01

Create a new file in src/routes/blog called POSTNAME.svx. Any file in this directory will be a new blog post.

These files are mdsvex files. This format is a combination of markdown syntax and svelte for creating components and javascript.

All your regular favourite markdown syntax is supported.

When being served, the files will use the filename as the post slug. So they’ll live over at blog/{slug}.
